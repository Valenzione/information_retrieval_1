# IR Assignment 1
# Installation
## Python dependencies
To run search engine you have to install these modules by running `pip install -r requirements.txt`
* pymongo
* lepl
* nltk
* pyQT (only for GUI)
## NLTK dependencies
NLTK may ask you to run nltk.download() to download specified package
# Running
## GUI
Run `gui.py`
## CLI
Run `cli.py`
